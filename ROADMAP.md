# Project Roadmap 🗺️

## Phase 1: Foundation (Weeks 1-2)
- [ ] Initialize Database schema (Users, Services, Transactions).
- [ ] Set up Authentication (JWT/Auth0) for Users and Admins.
- [ ] Build basic API boilerplate.

## Phase 2: Admin Power (Weeks 3-4)
- [ ] **Service Manager:** Interface to upload image, description, and price.
- [ ] **Transaction Table:** View all orders with "Pending/Completed" toggles.
- [ ] Image upload integration for tag previews.

## Phase 3: Customer Experience (Weeks 5-6)
- [ ] Product landing page showing services posted by admin.
- [ ] Multi-step checkout form:
    * Step 1: Personal Info (Name, Phone, Address).
    * Step 2: Vehicle Info (VIN, Car, Color).
    * Step 3: Delivery Address & Payment.
- [ ] Email notifications upon purchase.

## Phase 4: Security & Polish (Weeks 7+)
- [ ] VIN validation logic.
- [ ] Mobile responsive audit.
- [ ] Deployment to Vercel/DigitalOcean.
